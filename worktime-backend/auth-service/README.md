Аутентификация (авторизирует пользователей и хранит их токены)
Spring Boot + Spring Security. Access token 15 минут, refresh token 7 дней в httpOnly cookie . БД — PostgreSQL (таблица users, roles). Roles: admin - админ панель (управляет пользователями, мониторинг всего), team lead - тим лид команды (формирует команды, добавляет для них задачи) , user  - обычный сотрудник.
